## 📦 Fichiers à fournir (mets-les dans un dossier `/assets` à la racine du repo)

Tu peux partir d’images libres / fan-arts librement réutilisables, ou créer tes propres visuels (idéal). Noms & tailles ci-dessous :

1) **`/assets/bleach-banner.gif`** — *Banner animé principal*  
   - **Taille conseillée** : 1500×400 à 1920×480  
   - **Style** : fond noir/orange, silhouettes (ex: Ichigo/Zangetsu) + particules “reiatsu”  
   - **Poids** : < 4–6 MB si possible

2) **`/assets/pfp-hollow-mask.png`** — *Avatar stylé masque hollow (optionnel)*  
   - **Taille** : 256×256 (transparent, bord doux)

3) **`/assets/badge-soul-society.png`** — *Petit badge/écusson (optionnel)*  
   - **Taille** : ~300×300 (fond transparent)

4) **`/assets/reiatsu-left.gif`** et **`/assets/reiatsu-right.gif`** — *Colonnes d’énergie animées*  
   - **Taille** : ~120×300 (vertical, loop)

5) **`/assets/divider.svg`** — *Séparateur “reiatsu”*
